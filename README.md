# ACT 闪念簿

> 打开即写，快速捕捉想法，自动存入每日日志或任意项目笔记。

一个专为 **手机端 + 电脑端** 同时设计的 Obsidian 速记插件。核心体验是「打开 → 输入 → 保存」，中间无干扰。

---

## 功能特性

- **两种保存模式**：写入每日日志，或写入单一项目笔记
- **草稿自动保存**：输入过程自动缓存，关闭页面也不会丢
- **格式工具栏**：支持 `#`、加粗、项目符号、附件
- **聚焦模式**：隐藏下方历史，只保留编辑器专注输入
- **历史预览**：下方显示最近随想，支持跳转、编辑、删除
- **项目笔记切换**：单一笔记模式下可直接切换写入目标
- **项目笔记新建**：可在顶部直接新建笔记并加入可选项目
- **启动自动打开**：支持电脑端和手机端分别配置；手机端会等 Obsidian 首屏完成后再打开

## 安装

### 手动安装

1. 前往 [Releases](../../releases) 下载最新版本的 `main.js`、`manifest.json`、`styles.css`
2. 在 Vault 中创建目录 `.obsidian/plugins/act-capture/`
3. 将三个文件放入该目录
4. 在 Obsidian 设置 → 第三方插件中启用「ACT 闪念簿」

### 从源码构建

```bash
git clone https://github.com/owner/act-capture
cd act-capture
npm install
npm run build
```

将生成的 `main.js`、`manifest.json`、`styles.css` 复制到 Vault 的插件目录即可。

## 使用

打开方式：
- 命令面板（`Cmd/Ctrl + P`）→ 搜索「打开 ACT 闪念簿」
- 工具栏图标（message-square-plus）
- 在设置中开启「手机端/电脑端启动时自动打开」

启动速度：
- 手机端自动打开会做短暂延迟，让 Obsidian 先完成自身启动
- 输入编辑器会先显示，最近随想会稍后加载，避免历史预览阻塞首屏

编辑器操作：
- `Cmd/Ctrl + Enter`：保存
- `Cmd/Ctrl + B`：加粗
- `#` 按钮：插入 `#`
- 列表按钮：插入 `- `
- 附件按钮：把文件保存到当前目标笔记同目录的 `附件` 文件夹，并插入 `[[链接]]`

单一笔记模式：
- 顶部下拉框用于切换当前写入的项目笔记
- 「新建笔记」会在当前“项目随想文件夹”下创建一篇新的 Markdown 笔记
- 创建完成后会自动切换为当前写入目标

聚焦模式：
- 点击右上角「聚焦」进入
- 会隐藏项目选择和下方历史预览
- 点击右上角「退出」或按 `Esc` 退出

## 设置说明

通用设置：
- **手机端启动时打开**：开启后，手机端打开 Obsidian 会直接进入 ACT 闪念簿
- **电脑端启动时打开**：开启后，电脑端打开 Obsidian 会直接进入 ACT 闪念簿
- **保存模式**：选择「每日日志」或「单一笔记」

每日日志模式：
- **日志文件夹**：每日日志存放目录；留空时保存到 Vault 根目录
- **日志文件名格式**：支持 `YYYYMMDD`、`YYYY-MM-DD`，也兼容 `{date}`、`{weekday}` 等变量
- **日志模板路径**：新建当日日志时使用的模板；留空则使用插件内置内容
- **保存标题**：随想会写入到这个标题下面，例如 `每日随想`

单一笔记模式：
- **项目随想文件夹**：顶部项目选择和“新建笔记”都基于这个文件夹
- **随想二级标题格式**：每条随想会以 `##` 标题写入，支持 `YYYYMMDD HH:mm` 等格式
- **单一笔记路径**：当项目随想文件夹里没有可选笔记时，使用这篇笔记作为备用写入目标

当前推荐配置：
- 如果你要按天记录：使用「每日日志」模式
- 如果你要按项目沉淀：使用「单一笔记」模式，并设置好“项目随想文件夹”
